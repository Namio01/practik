1) В поле запрос Вы можете вводить "GET" и "POST" после чего вы должны нажать на кнопку "сортировать по запросу"
2) при нажатии на кнопку "сортировать по IP адресам" у вас появится файл с отсортированными по IP адресам данными 
3) при нажатии на кнопку "открыть базу данных" вы сможете посмотреть информацию о db файле
4) при нажатии на кнопку "удалить базу данных" база данных будет удалена
